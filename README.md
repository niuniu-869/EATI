# EATI 食格测试

> MBTI 测不准你的灵魂，但 EATI 能测出你的胃。

一个开源的娱乐性食格测试，厦门大学美食协会出品，致敬 B 站 UP 主 [@蛆肉儿串儿](https://space.bilibili.com/417038183) 的 SBTI 人格测试。

## 在线体验

👉 [点击开始测试](https://niuniu-869.github.io/EATI/)

## 特性

- 🍜 **25 种食格** — 23 种标准人格 + 2 种隐藏/兜底人格（火锅教祖 / 食欲已离线）
- 📊 **6 个维度** — 胃 / 舌 / 眼 / 腿 / 局 / 魂，覆盖吃的所有可观察行为
- 🎯 **曼哈顿距离匹配** — 基于 6 维向量的离谱匹配算法
- 🔥 **隐藏彩蛋** — 选"火锅能解决 99% 的情绪问题"强制触发 HOT-T 火锅教祖
- 📱 **移动端优先** — 响应式设计，手机截图发朋友圈刚刚好
- 🔧 **易于定制** — 数据与代码分离，改 JSON 就能 fork 一个你自己的测试

## 六维模型

| 维度 | 昵称 | 测什么 | L ← → H |
|---|---|---|---|
| **GUT** | 胃 | 食量与节奏 | 小鸟胃 ↔ 饕餮 |
| **TNG** | 舌 | 口味刺激度 | 清淡挑食 ↔ 生猛猎奇 |
| **EYE** | 眼 | 仪式与颜值 | 果腹主义 ↔ 调光拍照 |
| **LEG** | 腿 | 探索半径 | 楼下就行 ↔ 跨城打卡 |
| **CRW** | 局 | 社交模式 | 独狼 ↔ 整桌 MC |
| **SOUL** | 魂 | 吃的动机 | 生理续命 ↔ 情绪出口 |

## 定制你自己的测试

所有测试内容都在 `data/` 目录下，修改 JSON 文件即可定制，无需改代码。

### 修改题目

编辑 `data/questions.json`，每道题：

```json
{
  "id": "q_gut1",
  "dim": "GUT",
  "text": "你的题目场景",
  "options": [
    { "label": "选项 L", "value": 1 },
    { "label": "选项 M", "value": 2 },
    { "label": "选项 H", "value": 3 }
  ]
}
```

每个维度需要恰好 4 道题。

### 添加新人格

编辑 `data/types.json`，在 `standard` 数组添加：

```json
{
  "code": "YOUR",
  "pattern": "H-M-L-M-H-L",
  "cn": "你的人格名",
  "intro": "一句话金句",
  "desc": "疯癫长文..."
}
```

`pattern` 是 6 位 L/M/H，顺序：GUT-TNG-EYE-LEG-CRW-SOUL。

## 评分算法

1. **求和**：每维度 4 题分值相加（范围 4-12）
2. **分级**：≤6 → L（低），7-9 → M（中），≥10 → H（高）
3. **向量化**：L=1, M=2, H=3，生成 6 维向量
4. **匹配**：计算用户向量与每种人格的曼哈顿距离（最大 12）
5. **排序**：距离升序 → 精准命中降序 → 相似度降序
6. **特殊覆盖**：火锅教祖彩蛋 > 正常匹配 > 404 兜底（<60%）
